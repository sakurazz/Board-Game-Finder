About Board Game Finder

Welcome to our board game finder!
This game finder is designed by Calypso Leonard, Yingying Wang and Tresa Xavier for their Software Design Project at Carleton College.
Our site intends to help anyone and everyone find awesome new board games that make sense for their preferences, age, group size, and time constraints.
Not sure how to start?
Here's a rundown of what each feature of our search function means:

Category

Category describes the primary mechanism by which the game is played.
Co-operative, for instance, indicates a game where players work together towards a common goal, while Battle indicates a game that revolves around competition and war-like strategy. Categories are not all-encompassing, or mutually exclusive, as some games have multiple mechanisms.

Time

As we say on our homepage, this category indicates the maximum time you have available to play in minutes.
We will display all games which take up to the length you indicate.

Minimum Age

Not all games are appropriate for all players, this feature allows you to indiate the age of youngest person you'd like to be able to play the game.

Number of Players

Simply type in the number of players you would like to be able to play the game.

Ready to try it out?

You can use any combination of these features to search for games. If you don't care about one of them, simply leave that field blank.
